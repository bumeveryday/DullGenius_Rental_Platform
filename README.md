# 🎲 덜지니어스 보드게임 대여소 (DullG BoardGameRent) - Frontend

이 프로젝트는 덜지니어스 부원들이 보드게임을 쉽고 편리하게 대여/반납하고, 관리자가 이를 효율적으로 운영할 수 있도록 돕는 웹 서비스입니다. 
**React(Frontend)** 와 **Google Apps Script(Backend)** 를 연동하여 서버리스 환경에서 구축되었습니다.

---

## ✨ 주요 기능 (Features)

### 🧑‍💻 일반 사용자 (User)

**1. 보드게임 탐색 & 조회**
- **검색 & 필터**: 게임 이름, #해시태그, 인원수, 난이도, 카테고리(TRPG 등)로 원하는 게임을 빠르게 찾을 수 있습니다.
- **상세 정보**: 게임의 난이도, 장르, 추천 인원, 현재 대여 가능 상태를 확인할 수 있습니다.
- **트렌드**: '요즘 뜨는 게임' (대여/조회 급상승) TOP 5를 메인 화면에서 확인 가능합니다.
- **상황별 추천**: 관리자가 설정한 테마별(예: "커플 추천", "파티 게임") 추천 게임 목록을 원클릭으로 볼 수 있습니다.

**2. 대여 시스템**
- **방문 수령 예약 (Dibs)**: 원하는 게임을 '30분간 찜'할 수 있습니다. 30분 내에 동아리방에 방문하여 수령하면 됩니다.
- **간편 회원가입/로그인**: 
    - 대여 시 학번/이름/전화번호 입력만으로 즉시 대여가 가능합니다.
    - **자동 로그인**: 기존 회원이면 비밀번호 확인 후 즉시 대여가 진행됩니다.
    - **로그인 유지**: 한 번 인증하면 앱 종료 전까지 로그인 상태가 유지됩니다.

**3. 리뷰 & 커뮤니티**
- **리뷰 작성**: 대여한 게임에 대해 별점과 한줄평을 남길 수 있습니다. (로그인 필수)
- **리뷰 관리**: 본인이 작성한 리뷰를 삭제할 수 있습니다.
- **아쉬워요 (입고 요청)**: 대여 중인 게임이나 없는 게임에 대해 '아쉬워요'를 눌러 관리자에게 입고/반납 요청 수요를 보낼 수 있습니다.

**4. 마이페이지**
- **대여 내역 조회**: 본인의 현재 대여 목록과 반납 기한을 확인할 수 있습니다.

---

### 🛡️ 관리자 (Admin)

**1. 대시보드 (Dashboard)**
- **현황 파악**: 현재 대여 중인 게임 수, 연체 현황, 최근 활동 로그를 한눈에 볼 수 있습니다.
- **데이터 시각화**: 일별/월별 대여 통계 그래프를 제공합니다.

**2. 게임 관리 (Game Management)**
- **게임 추가**: 
    - **수동 추가**: 이름, 이미지, 장르 등을 직접 입력.
    - **BGG/네이버 검색 연동**: BoardGameGeek 또는 네이버 쇼핑 API를 통해 게임 정보를 자동으로 불러와 등록할 수 있습니다.
- **게임 수정/삭제**: 기존 게임의 정보를 수정하거나 목록에서 삭제할 수 있습니다.
- **상태 관리**: 분실/파손 등의 사유로 게임 상태를 변경(대여 불가 지정)할 수 있습니다.

**3. 대여/반납 관리**
- **현장 대여/반납 처리**: 관리자가 직접 유저의 아이디를 검색하여 대여 및 반납 처리를 수행할 수 있습니다.
- **일괄 처리**: 특정 유저가 빌린 모든 게임을 한 번에 반납 처리할 수 있습니다.

**4. 리뷰 & 유저 관리**
- **리뷰 삭제**: 부적절한 리뷰를 관리자가 강제로 삭제할 수 있습니다.
- **유저 목록**: 전체 회원 목록과 벌금/회비 납부 현황을 조회할 수 있습니다.

**5. 시스템 설정**
- **추천 버튼 관리**: 메인 화면의 '상황별 추천' 버튼(텍스트, 색상, 태그)을 자유롭게 커스터마이징할 수 있습니다.

## 🛠️ 기술 스택 (Tech Stack)

*   **Frontend**: React.js, React Router, CSS Modules
*   **Backend**: Google Apps Script (GAS)
*   **Database**: Google Sheets (NoSQL-like usage)
*   **Deployment**: Vercel / GitHub Pages (Frontend), Google Script Web App (Backend)

## 🚀 시작하기 (Getting Started)

### 1. 설치
```bash
npm install
```

### 2. 실행
```bash
npm start
```
`http://localhost:3000` 에서 실행됩니다.

### 3. 환경 설정 (.env)
```ini
REACT_APP_API_BASE_URL=https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec
```
